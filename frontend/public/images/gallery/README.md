# Achievement & Gallery Photos

**Put your uploaded photos in this folder.**

The website auto-detects images by filename. Use any of these names:

## Certificate (Hackathon)

- `certificate.jpg` / `certificate.png`
- Or any name containing: **cert**, **certificate**, **hackathon**

Examples: `hackathon-certificate.png`, `My Certificate.jpeg`

## Gifts & Awards

- `gifts.jpg` / `gifts.png`
- Or any name containing: **gift**, **gifts**, **award**, **prize**

Examples: `hackathon-gifts.jpg`, `prizes.png`

## After adding files

1. Save images here
2. Restart `npm run dev`
3. Refresh the browser

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`

Alternative folder: `frontend/src/assets/gallery/` (same naming rules)
